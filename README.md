# Space-Shooter-Game

유튜버 "코딩 알려주는 누나"의 영상을 참고하여 만든 레트로 우주슈팅게임

## 동작설명

* 키보드의 좌우키로 우주선을 조종하고 스페이스바로 총알을 발사할 수 있다.
* UFO가 1초간격으로 랜덤하게 내려오고 총알로 UFO를 격추시키면 SCORE가 올라간다.
* 우주선이 화면 바닥에 닿으면 게임오버화면이 뜨며 게임이 멈춘다.

## 사용스택

html, css, javascript
